# Radha Rani — Portfolio

A single-page portfolio site built from your resume, ready to host for free on GitHub Pages.

## Deploy it in 5 minutes

1. Create a new GitHub repository. Name it exactly `Radha-byte.github.io` (this special name makes GitHub host it automatically at that URL — replace `Radha-byte` with your GitHub username if different).
2. Upload `index.html` to the root of that repository (drag-and-drop works fine on github.com, or use `git add / git commit / git push`).
3. Go to the repo's **Settings → Pages**. Under "Build and deployment," set Source to **Deploy from a branch**, branch `main`, folder `/root`. Save.
4. Wait 1–2 minutes, then visit `https://Radha-byte.github.io`. Your site is live.
5. Add that link to the "Featured" section (or the URL field) on your LinkedIn profile so it's the first thing people see.

If you'd rather keep it inside an existing repo instead of a dedicated `username.github.io` one, that also works — just enable Pages on that repo and it'll be hosted at `https://Radha-byte.github.io/repo-name/`.
